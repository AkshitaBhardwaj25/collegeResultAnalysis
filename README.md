# College Exam Result Analysis System

## Project Overview
This project is a full-stack **DBMS system** that analyzes and visualizes college exam results.  
It connects **Python + MySQL** for backend and uses **Streamlit** for frontend dashboards.

---

## Features
- Automatic database and table creation in MySQL  
- Trigger and stored procedures for grade assignment and GPA/CGPA calculation  
- Student, Course, Faculty, and Result management  
- Random data generation using Faker library  
- Analytics and visual reports:
  - Semester-wise toppers
  - Subject-wise averages
  - GPA/CGPA calculation
  - Interactive charts
